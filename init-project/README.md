## About Init-Project
This is template for create a new laravel 11 project
### Required System
* VS-Code or etc.
* XAMPP
* Composer
* MySql
### Step Create Project
* composer create-project laravel/laravel [project-name]
* migrate database - php artisan migrate
* seed database - php artisan db:seed || php artisan db:seed --class=[classname]
